# LocalhostLink
Script bas simple untuk meremot loaclhost anda menjadi link publik

## If your server system: Debian/Ubuntu
```bash
apt-get update
git clone https://github.com/BM-TechID/LocalhostLink.git
cd LocalhostLink
chmod +x *.sh
./install.sh

#run
./link.sh
```

## Change Bot Token & Your Id Telegran
```bash
cd Localhost
nano bot.sh

#find
TOKEN_BOT=""
CHAT_ID=""


ctrl+x, y, and enter.
done
```
